📖 Правила торговли
Обозначения дней
T0 (сигнальный день) — день, когда выполнены все условия сетапа (на закрытии)
D+1 — первый торговый день после T0, день входа
D+k — k-й торговый день после T0
D+5 — финальный принудительный выход (time stop)
Базовые метрики
ATR14 = Wilder ATR на 14 дневных барах
ATRp = ATR14 / Close × 100%
SMA_N(X) = простое скользящее среднее X за N дней
EMA_N(X) = экспоненциальное скользящее среднее X за N дней
RSI14(X) = RSI Уайлдера от X за 14 дней
RSI2(X) = RSI Уайлдера от X за 2 дня
ADV20 = среднее ($ оборот = Close × Volume) за 20 дней
IBS = (Close − Low) / (High − Low)
Range = High − Low
Return_N = Close_t / Close_{t-N} − 1
Базовый риск на сделку
RiskPerTrade_$ = 1% от капитала (фиксированная константа)
StopDistance_$ = |Entry − StopPrice|
Shares = floor( RiskPerTrade_$ / StopDistance_$ )
PositionSize_$ = Shares × Entry
MaxPositionSize = 25% капитала (ограничение для low-vol сетапов)
🚫 Общие запреты (для всех стратегий)
1. Нет сигнала, если на T0 объявлены earnings в течение D+1 или D+2 (кроме PEAD)
2. Нет сигнала, если VIX ≥ 35 на закрытии T0 (экстремальная волатильность)
3. Нет сигнала, если Range > 2 × ATR14 (news-driven bar)
4. Нет сигнала, если цена < $5 или ADV20 < $20M
5. Максимум 5 одновременных позиций
6. Не открывать две одинаково-направленные позиции в одном секторе (GICS L1) в один день
📊 IBS Mean Reversion
Классический mean-reversion на откате в трендовой акции. IBS используется как фильтр качества отката.
Market Regime Filter
LongRegime = SPY_Close_T0 > SMA200(SPY)_T0 → разрешены только long
ShortRegime = SPY_Close_T0 < SMA200(SPY)_T0 → разрешены только short
Если SPY ровно на SMA200 (±0.1%) — сделки запрещены
🟢 Условия сигнала LONG (T0)
1. Close_T0 > SMA200_T0
2. SMA200_T0 > SMA200_{T0−20} (тренд растёт)
3. IBS_T0 < 0.20
4. RSI2_T0 < 10
5. Range_T0 < 2 × ATR14_T0
6. LongRegime = true
🔴 Условия сигнала SHORT (T0)
1. Close_T0 < SMA200_T0
2. SMA200_T0 < SMA200_{T0−20} (тренд падает)
3. IBS_T0 > 0.80
4. RSI2_T0 > 90
5. Range_T0 < 2 × ATR14_T0
6. ShortRegime = true
Правила входа (D+1)
Entry = Open_{D+1} (market-at-open)
Отмена входа:
• LONG: если Open_{D+1} ≥ Close_T0 × 1.02 → гэп ≥2% вверх
• SHORT: если Open_{D+1} ≤ Close_T0 × 0.98 → гэп ≥2% вниз
Стоп и размер позиции
StopDistance = min( 1.5 × ATR14_T0, 0.06 × Entry )
LONG: StopPrice = Entry − StopDistance
SHORT: StopPrice = Entry + StopDistance
Shares = floor( Capital × 0.01 / StopDistance )
Правила выхода
Target 1 = Entry ± 1.0 × ATR14 → закрыть 50%, стоп → безубыток
Target 2 = Entry ± 2.0 × ATR14 → закрыть остаток
Trailing (после Target 1) = Close − 1.0 × ATR14 (подтягивается только вверх для LONG)
T0-day check: если High_{D+1} − Entry < 0.5 × ATR → выход по Close_{D+1}
Time stop: на закрытии D+5 — принудительный выход остатка
👁 Правила формирования списка
Universe требования для всех стратегий
Базовые требования
1. Акции индексов S&P 500 и Nasdaq-100
2. Close ≥ $10
3. ADV20 ≥ $20M
4. Историческая цена доступна за 250+ торговых дней
5. Сектор указан (GICS Level 1)
Поля инструмента
ticker — символ (обязательно)
sector — GICS сектор L1 (обязательно)
atr — ATR% (обязательно для расчёта стопа и целей)
cap — капитализация (Large/Mid/Small)
earningsDate — дата ближайших earnings
cooldownDate — дата окончания кулдауна после earnings
notes — заметки
Blackout правила
• Earnings blackout: −2 дня до earningsDate
• Cooldown: до cooldownDate после earnings
• Инструмент с blackout/cooldown не участвует в сканировании
